## System Prompt Optimization

### Analytical Thinking

```
I need you to think step-by-step through this problem.
Please:
1. Break down the requirements clearly
2. Consider multiple approaches
3. Evaluate trade-offs and implications
4. Provide a reasoned recommendation
5. Explain your reasoning process

Focus on thorough analysis and clear communication.
```

### Code Generation

```
I need you to write high-quality code that is:
- Correct and efficient
- Well-documented and readable
- Following best practices
- Properly tested
- Secure by design

Please provide:
- Complete implementation
- Explanation of approach
- Test cases where appropriate
- Documentation comments
```

### Problem Solving

```
I have a complex problem that requires careful analysis.
Please help me by:
- Understanding the core issue
- Identifying constraints and requirements
- Exploring different solution approaches
- Recommending the best path forward
- Considering potential risks and mitigation strategies
```

## Context Injection Patterns

### For Technical Documentation

```
I'm working on technical documentation that needs to be:
- Clear and comprehensive
- Accessible to different skill levels
- Well-structured and organized
- Include practical examples
- Follow documentation best practices

Please help me create content that serves these goals.
```

### For Architecture Decisions

```
I need to make an architectural decision for a software project.
Please help me:
- Analyze the requirements and constraints
- Consider multiple architectural approaches
- Evaluate pros and cons of each option
- Recommend the most suitable approach
- Document the decision rationale
```

## Response Formatting

### Structured Analysis

- Use clear headings and sections
- Provide numbered steps for processes
- Include bullet points for key information
- Use tables for comparisons
- Add diagrams when helpful

### Code Examples

- Provide complete, working code
- Include necessary imports and setup
- Add comprehensive comments
- Show usage examples
- Include error handling
