## رسائل المستخدم

### رسائل التأكيد

```
'هل أنت متأكد من حذف هذا العميل؟'
'سيتم حذف جميع الفواتير المرتبطة. هل تريد المتابعة؟'
'هل تريد حفظ التغييرات قبل الخروج؟'
'هل تريد إلغاء العملية؟'
```

### رسائل الخطأ

```
'رقم الهاتف يجب أن يبدأ بـ 05 ويتكون من 10 أرقام.'
'البريد الإلكتروني غير صحيح. يرجى التحقق من الصيغة.'
'لا يمكن حذف هذا العميل لأنه مرتبط بفواتير موجودة.'
'حدث خطأ أثناء حفظ البيانات. يرجى المحاولة مرة أخرى.'
```

### رسائل النجاح

```
'تم إضافة العميل بنجاح.'
'تم حفظ الفاتورة بنجاح.'
'تم تصدير الفاتورة كملف PDF بنجاح.'
'تم تحديث البيانات بنجاح.'
```
